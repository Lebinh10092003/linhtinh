# Poster lớp học số - THCS Nguyễn Du V2

Đây là bản dựng lại bằng code với bố cục bám sát mẫu hơn bản V1.

## File chính

- `src/render_poster.py`: mã nguồn dựng poster bằng Python/Pillow.
- `dist/lop-hoc-so-nguyen-du-v2.png`: ảnh PNG sẽ được xuất khi chạy script.
- `requirements.txt`: thư viện cần cài.

## Cách xuất lại ảnh

```bash
pip install -r requirements.txt
python src/render_poster.py
```

## Cách sửa nội dung

Mở `src/render_poster.py`, sửa các đoạn chữ, màu sắc, vị trí hoặc kích thước rồi chạy lại lệnh xuất ảnh.

Ghi chú: đây là bản dựng lại bằng code, không phải file thiết kế gốc Canva/Figma. Vì vậy có thể sửa bằng code và xuất ảnh, nhưng không phải toàn bộ ảnh được tách thành layer như file thiết kế gốc.
