# Poster lớp học số - đúng mẫu

Bản này dùng chính ảnh mẫu làm nền nên hiển thị đúng mẫu gốc.

## File chính

- `index.html`: mở trực tiếp bằng trình duyệt, xem đúng mẫu và bấm tải JPG/PNG hoặc in/lưu PDF.
- `dist/lop-hoc-so-nguyen-du-exact.png`: ảnh PNG đúng mẫu.
- `assets/poster-reference-q85.jpg`: ảnh nén dùng trong HTML.

## Lưu ý quan trọng

Ảnh mẫu hiện là ảnh phẳng. Vì vậy có thể làm đúng mẫu 100% về mặt hiển thị bằng cách dùng ảnh gốc, nhưng không thể sửa từng chữ/icon như file Canva/Figma nếu không có file thiết kế gốc.

Muốn vừa đúng mẫu vừa sửa từng phần, cần dựng lại thủ công theo layer trong Canva/Figma/PowerPoint.
